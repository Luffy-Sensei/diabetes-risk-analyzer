# 🩺 Diabetes Risk Analyzer
A simple Android application that predicts diabetes risk levels based on user-input health metrics.

## ✨ Features
- User Registration & Login
- Input health data (BMI, Age, Glucose levels, etc.)
- Risk analysis results page

## 🛠️ Tech Stack
- **Language:** Java
- **Framework:** Android SDK
- **Database:** Firebase 

## 🚀 How to Run
1. Clone the repo: `git clone https://github.com/Luffy-Sensei/diabetes-risk-analyzer.git`
2. Open the project in **Android Studio**.
3. Build and run on an emulator or physical device.
